Gestión de iventario(actividad concesionario)

Descripción:

EL programa tine como objetivo servir como vase de datos consultable y esitable para un concesionario en el que el usuario podra introducir nuevos veiculos(productos) mas una serie de atributos predefinidos, ademas de modificar el nombre y atributos de los productos registrados, eliminarlos o revisar la disponivilidad de los productos en todo momento.

Requisitos: Tener instalado Python 3.x

Instrucciones de uso:

El programa es realmente sencillo, una vez ejecutado se vera el menu de de obciones, primero que nada es requerido añadir un primer veiculo al inventario, por lo que se debe marcar la obcion 1 de la forma que indica el propio programa, e ir introduciendo los atributos del objeto a la par de que el programa los pida.

Tras esto, ya se pueden usar las demas obciones que son modificar un cohe ya añadidio, eliminarlo, o revisar los coches que hay en el inventario.
Cabe recalcar que puede añadir tantos coches como desee, aunque solo puede añadir un coche de un modelo especifico, si intenta añadir otro coche que comparta marca y modelo con uno ya registrado, el programa le indicara que ya hay un coche con dichas marca y modelo en el inventario y que no se puede añadir mas de uno.

Para cada obcion hay un numero asignado en el menu.

En el caso de la obcion de revisar el inventario, el programa le dara directamente una lista de todos los coches en el inventario y sus atributos, y si no hay ningino, se lo indicara.

En el caso de que elija la obcion de modificar un producto, el programa le pedira la marca y modelo del coche a modificar, si el coche esta en el inventario, el programa le pedira de forma ordenada los nuevos atributos para el coche, en caso de que no este se lo indicara.

En el caso de que elija eliminar un coche, el programa le pedira la marca y el modelo de este, en caso de que este en el inventario, el programa lo eliminara y le indicara que se a eliminado exitosamente, en caso de que no este presente, tambien se lo indicara.
